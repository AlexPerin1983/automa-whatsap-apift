🔘 Botões Interativos (Anti-Spam)
Cada follow-up incluirá 3 botões do modelo principal salvo em Disparos de botoes.
Modelo atual: ✅ Quero ver | ⏰ Depois | 🚫 Não tenho interesse
• 1º botão — interesse / continuar conversa
• 2º botão — adia o próximo follow-up
• 3º botão — lista negra para contatos reais, mas permanece válido em números de teste
⚠️ O botão "Sair" é essencial: quem não encontra uma saída fácil denuncia como spam. Com o botão visível, o lead se sente seguro e não bloqueia.
